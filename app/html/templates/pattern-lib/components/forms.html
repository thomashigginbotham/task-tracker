Forms

Components that assist with input methods are listed below.

Tracker Form Block {{instructionText}}



		


		
A Form Block contains Form Row and Button Row children. Additional styling is applied to Button Rows when placed inside a Form Block.

	


	


	

		
Tracker Form Row {{instructionText}}


		


		


		
A Form Row can display any number of group boxes in a horizontal layout. The form fields within the group boxes can be stretched to fill the full width by using the stretch attribute. The flex attribute is available for setting the flex shorthand values for each group box.


		

			
				

					Attribute
					Description
				

			

			
				

					stretch
					A boolean attribute to stretch the form fields to fill the full row. Note: the flex attribute must be set for the stretch attribute to take effect.
				

				

					flex
					An array of flex values for each group box in the row.
				

			
		

	


	


	

		
Tracker Search Block {{instructionText}}


		


		


		
The Search Block makes use of a Group Block to create a simple search box.


		

			
				

					Attribute
					Description
				

			

			
				

					uri
					The URI of the form submission handler.
				

			
		

	


	


	

		
Tracker Toggle Fields {{instructionText}}


		


		


		
The Toggle Fields component will display a list of check boxes or radio buttons in an inline format. Toggle Fields extend the native <ul> element.
