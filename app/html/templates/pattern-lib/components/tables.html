Tables

Data represented in rows and columns should make use of table components.

Sorted Table {{instructionText}}



		


		
A sorted table extends the HTML table element and provides headings that may be used to sort their respective columns.

		
Columns are sorted based on the contents of each cell by default. To create custom sort values, add a
			data-sort-value to the
			<td> tag.


		

			
				

					Attribute
					Description
				

			

			
				

					columns
					An array of table headers.
				

				

					no-sort
					An array of column indexes that should not allow sorting.
				

				

					widths
					An array of widths for each header.
				

				

					aligns
					An array of alignment values (
						left,
						center, or
						right) for each header.
					
				

				

					default-sort-col
					The index of the column to be sorted by default.
				

				

					default-sort-order
					The direction of the default sort. Can be either
						asc (default) or
						desc.
