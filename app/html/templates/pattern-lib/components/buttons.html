Buttons

Buttons are elements that are tapped or clicked to preform an action.

Tracker Icon Button {{instructionText}}



		


		
An Icon Button allows an icon to be placed before the button text.


		

			
				

					Attribute
					Description
				

			

			
				

					uri
					The URI for the hyperlink.
				

				

					icon
					An icon name from the Font Awesome font.
				

				

					caution
					A boolean attribute that changes the button background to orange.
				

			
		

	


	


	

		
Tracker Button Row {{instructionText}}


		


		


		
When more than one button is needed, or if a button needs to be paired with a message, a button row helps to group them together.

		
A button row should contain at least one type of Button component. If a message should be displayed, it should be placed within a <p> tag.


		

			
				

					Attribute
					Description
				

			

			
				

					joined
					A boolean attribute that removes extra space between buttons.
