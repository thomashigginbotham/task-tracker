General Layout

Important parts of common page layout are included here. These components are found in the app/templates directory.

To add components to this area, edit the files in the app/templates/pattern-lib/components directory.

Task Tracker App {{instructionText}}



		


		
The task tracker app component acts as a master layout for the application.

		
The <task-tracker-app> tag has no available attributes and
			expects to have three <group-pane> elements and a
			<footer-panel> element as descendents. The class
			names and order of these child elements are important. Reference the
			HTML in the sample above for more detail.

	


	


	

		
Group Pane {{instructionText}}


		


		


		
Group panes, created with the <group-pane> tag, are
			expected as direct descendents of an <task-tracker-app> tag.
			There are three types of group panes: menu, detail, and properties;
			each determined by their respective class names: menu-pane,
			detail-pane, and properties-pane, respectively.

		
group panes have two available attributes. The heading
			attribute allows for the addition of a heading to the top of the
			pane. The icon attribute can be set to an icon name
			available in the Font Awesome font family, and will cause that icon
			to appear to the left of the heading.

		
A menu may be added to the paneâ€™s header by placing a
			menu block component within
			group-pane-menu tags.


		

			
				

					Attribute
					Description
				

			

			
				

					heading
					A heading to appear at the top of the pane.
				

				

					color
					A CSS color value to use for the heading background.
				

				

					icon
					An icon name from the Font Awesome font.
				

			
		

	


	


	

		
Footer Pane {{instructionText}}


		


		


		
The footer pane is added to the bottom of the task tracker app. It is
			used to display a status message and version/license information.


		

			
				

					Attribute
					Description
				

			

			
				

					status
					Possible values are online and offline.
				

				

					version
					The current version of Task Tracker.
				

			
		

	


	


	

		
Form Row {{instructionText}}


		


		


		
A form row can display any number of group boxes in a horizontal layout. The form fields within the group boxes can be stretched to fill the full width by using the stretch attribute. The flex attribute is available for setting the flex shorthand values for each group box.


		

			
				

					Attribute
					Description
				

			

			
				

					stretch
					A boolean attribute to stretch the form fields to fill the full row.
				

				

					flex
					An array of flex values for each group box in the row.
