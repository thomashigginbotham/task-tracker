General Layout

Important parts of common page layout are included here. To add components to this area, edit the files in the app/templates/pattern-lib/components directory.

Tracker App {{instructionText}}



		


		
The Tracker App component acts as a master layout for the application.

		
The <task-tracker-app> tag has no available attributes and expects to have four child elements: tracker-menu-pane, tracker-detail-pane, tracker-property-pane, and tracker-footer-pane, in that order.

		
All Polymer components used within the app are imported from the tracker-app definition file.

	


	


	

		
Tracker Group Pane {{instructionText}}


		


		


		
Group Panes are intended only for use by Menu Panes, Detail Panes, and Property Panes. These three panes have many similarities, so they are basically wrappers for Group Panes.


		

			
				

					Attribute
					Description
				

			

			
				

					heading
					A heading to appear at the top of the pane.
				

				

					font-color
					A CSS color value to use for the heading forecolor.
				

				

					bg-color
					A CSS color value to use for the heading background.
				

				

					icon
					An icon name from the Font Awesome font.
				

			
		

	


	


	

		
Tracker Group Pane Menu {{instructionText}}


		


		


		
A menu can be inserted into the header of a Group Pane by adding a Group Pane Menu component. The Group Pane Menu expects a single Menu Block component as a child.

	


	


	

		
Tracker Menu Pane {{instructionText}}


		


		


		
The Menu Pane is a wrapper around a Group Pane, so it provides the same features and attributes with different styling. It is intended to provide navigation for the application.


		

			
				

					Attribute
					Description
				

			

			
				

					heading
					A heading to appear at the top of the pane.
				

				

					font-color
					A CSS color value to use for the heading forecolor.
				

				

					bg-color
					A CSS color value to use for the heading background.
				

				

					icon
					An icon name from the Font Awesome font.
				

			
		

	


	


	

		
Tracker Detail Pane {{instructionText}}


		


		


		
The Detail Pane is a wrapper around a Group Pane, so it provides the same features and attributes with different styling. This pane should contain the primary content for the application.


		

			
				

					Attribute
					Description
				

			

			
				

					heading
					A heading to appear at the top of the pane.
				

				

					font-color
					A CSS color value to use for the heading forecolor.
				

				

					bg-color
					A CSS color value to use for the heading background.
				

				

					icon
					An icon name from the Font Awesome font.
				

			
		

	


	


	

		
Tracker Property Pane {{instructionText}}


		


		


		
The Property Pane is a wrapper around a Group Pane, so it provides the same features and attributes with different styling. Property Panes should contain ancillary content for the Detail Pane.


		

			
				

					Attribute
					Description
				

			

			
				

					heading
					A heading to appear at the top of the pane.
				

				

					font-color
					A CSS color value to use for the heading forecolor.
				

				

					bg-color
					A CSS color value to use for the heading background.
				

				

					icon
					An icon name from the Font Awesome font.
				

			
		

	


	


	

		
Tracker Footer Pane {{instructionText}}


		


		


		
The Footer Pane is added to the bottom of the Tracker App. It is
			used to display a status message and version/license information.


		

			
				

					Attribute
					Description
				

			

			
				

					status
					Possible values are online and offline.
				

				

					version
					The current version of Task Tracker.
				

			
		

	


	


	

		
Tracker Drawer {{instructionText}}


		


		


		
The Drawer can be added to the top of a Detail Pane and is used for displaying information or forms pertaining to the pane. It is hidden by default, but can be shown by changing its visible attribute or by setting the is-drawer-toggle attribute of a Menu Item.


		
It may contain any HTML content.


		

			
				

					Attribute
					Description
				

			

			
				

					heading
					A heading to appear above the content.
				

				

					visible
					A boolean attribute to indicate if the drawer is visible.
