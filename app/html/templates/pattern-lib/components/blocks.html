Blocks

Block components help with grouping related content together.

Group Block {{instructionText}}



		


		
A group block is a very basic block that allows for a heading to be placed above the grouped content.


		

			
				

					Attribute
					Description
				

			

			
				

					heading
					A heading to appear above the content.
				

			
		

	


	


	

		
Menu Block {{instructionText}}


		


		


		
Similar to group blocks, menu blocks act as a wrapper for content and provide a heading above. However, a menu block is intended as a form of navigation and may only contain menu-item children.


		

			
				

					Attribute
					Description
				

			

			
				

					heading
					A heading to appear above the menu.
				

				

					layout
					A value of vertical (default) will create a vertical menu. A value of horizontal will create a horizontal menu.
				

			
		


		
Menu Item


		
A menu item is an individual hyperlink within a menu block. Menu items expose three attributes:


		

			
				

					Attribute
					Description
				

			

			
				

					uri
					The URI for the hyperlink.
				

				

					icon
					An icon name from the Font Awesome font.
				

				

					active
					A boolean attribute to indicate a currently active menu item
				

			
		

	


	


	

		
Notice Block {{instructionText}}


		


		


		
A notice block can be added to the top of a group pane and is used for displaying notifications or for displaying a form to allow users to make changes pertaining to the group pane.


		
It may contain any HTML content.


		

			
				

					Attribute
					Description
				

			

			
				

					heading
					A heading to appear above the content.
				

			
		

	


	


	

		
Paper Block {{instructionText}}


		


		


		
A paper block is similar to a group block in that it has a heading
			and allows any type of content. The content itself, however, is
			displayed within a box with a shadow, creating a three-dimensional
			effect.


		

			
				

					Attribute
					Description
				

			

			
				

					heading
					A heading to appear above the content.
