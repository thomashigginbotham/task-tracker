Blocks

Block components help with grouping related content together.

Tracker Group Block {{instructionText}}



		


		
A group block is a very basic block that allows for a heading to be placed above the grouped content.


		

			
				

					Attribute
					Description
				

			

			
				

					heading
					A heading to appear above the content.
				

			
		

	


	


	

		
Tracker Menu Block {{instructionText}}


		


		


		
Similar to Group Blocks, Menu Blocks act as a wrapper for content and provide a heading above. However, a Menu Block is intended as a form of navigation and may only contain tracker-menu-item children.


		

			
				

					Attribute
					Description
				

			

			
				

					heading
					A heading to appear above the menu.
				

				

					font-color
					A CSS color value to use for the heading forecolor.
				

				

					layout
					A value of vertical (default) will create a vertical menu. A value of horizontal will create a horizontal menu.
				

			
		

	


	


	

		
Tracker Menu Item


		
A Menu Item is an individual hyperlink within a Menu Block (see Tracker Menu Block above for an example). Menu Items expose the following attributes:


		

			
				

					Attribute
					Description
				

			

			
				

					uri
					The URI for the hyperlink.
				

				

					font-color
					A CSS color value to use for the heading forecolor.
				

				

					icon
					An icon name from the Font Awesome font.
				

				

					active
					A boolean attribute to indicate a currently active menu item
				

				

					is-drawer-toggle
					A boolean attribute that, when present, causes the Menu Item to act as a toggle for opening and closing the Drawer component.
				

			
		

	


	


	

		
Tracker Paper Block {{instructionText}}


		


		


		
A Paper Block is similar to a Group Block in that it has a heading
			and allows any type of content. The content itself, however, is
			displayed within a box with a drop shadow, creating a three-dimensional
			effect.


		

			
				

					Attribute
					Description
				

			

			
				

					heading
					A heading to appear above the content.
				

				

					important
					A boolean attribute to indicate that the block should be prominently featured.
