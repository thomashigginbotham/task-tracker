Page Sections

Important parts of common page layout, such as the header, footer, and sidebars are included here. These components are found in the app/partials directory.

To add components to this area, edit the files in the app/partials/pattern-lib/components directory.

Site Header {{instructionText}}



	


	
This is a sample description for a header. Modify it to match your actual header.

	
The header consists of a logo on the left and a search field on the right. The color theme is represented at the top using a thin band of color boxes. On smaller viewports, the search field will move below the logo.
